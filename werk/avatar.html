Avatar: The Game

Opdracht

Ontwerp en maak een website met behulp van HTML, CSS, en Javascript. De website moet functionaliteiten bevatten geïnspireerd door Tamagotchi.

Proces

Tijdens dit hele schooljaar heb ik geprobeerd onderwerpen te kiezen die ik zelf interessant vind, voor dit werk geldt dit ook. Ik ben een ontzettende fan van Avatar: The Last Airbender en ik heb altijd de serie als spel willen spelen. Voor deze opdracht greep ik dus mijn kans.
Eigenlijk van het begin was ik vrij ambitieus in mijn idee, hierdoor was het coderen af en toe vrij lastig. Wat ik vooral moeite mee had is het werkende maken dat Javascript de data zou onthouden die verzameld wordt zodra de gebruiker de website opent. In het spel moet je namelijk trainen om dan in een andere pagina deze informatie toe te passen. Daarnaast was het ook een werkje om te zorgen dat de gebruiker een “move” kon gebruiken door op een specifieke knop op hun toetsenbord te klikken.

Reflectie

Als ik terugkijk zou ik zelf nog verder mijn concept willen uitwerken. Vooral een vechtsysteem in het spel maakt het wel compleet. Eigenlijk meer functionaliteiten zoals het kiezen in welk element je het spel wilt spelen.
Voor de rest is vooral code opknappen en de kennis die ik nu heb gebruiken om de code nog efficiënter te kunnen laten werken.
